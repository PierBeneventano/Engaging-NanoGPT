# Research Prompt Workspace

Use this folder to store and iterate on system prompts for ML theory research.

## Structure

- `incoming/`: Draft or raw system prompts.
- `optimized/`: Refined prompts after improvements.
- `template/`: Reusable scaffolds and quality checklist.

## Workflow

1. Create a new file in `incoming/` (for example: `2026-03-02-ml-theory-v1.md`).
2. Write your initial system prompt.
3. Use `template/OPTIMIZATION_CHECKLIST.md` to improve it.
4. Save improved version in `optimized/` with an incremented version name
   (for example: `2026-03-02-ml-theory-v2.md`).
5. Keep short notes at the bottom: what changed and why.

## Naming convention (recommended)

`YYYY-MM-DD-topic-vN.md`

Example: `2026-03-02-generalization-bounds-v1.md`
